# 🔐 Password Generator (Streamlit)

A secure password generator developed using Python and Streamlit. Users can generate strong random passwords with customizable options.

## 🚀 Features

- Select password length
- Include uppercase letters
- Include lowercase letters
- Include numbers
- Include special characters
- Generate secure passwords instantly

## 🛠️ Technologies Used

- Python
- Streamlit
- Random
- String

## 📂 Project Structure

```
Password_Generator_Streamlit/
│
└── password_generator.py
```

## ▶️ How to Run

Install Streamlit

```bash
pip install streamlit
```

Run

```bash
streamlit run password_generator.py
```
